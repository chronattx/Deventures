## Идея игры
Кратко, Deventures (Depressing adventures) - игра-платформер в жанре RPG с видом Top-Down. Игрок проходит уровни,
сражаясь и учавствуя в диалогах, и каждый элемент игры заставляет его испытывать душевные муки.
